# Mini-projet Fireworks

## Description

Une simple application web qui affiche des feux d'artifice colorés lorsqu'on clique sur l'écran.

## Comment l'utiliser

1.  Assurez-vous que les fichiers `index.html`, `styles.css`, et `app.js` sont dans le même dossier.
2.  Ouvrez le fichier `index.html` directement dans votre navigateur web préféré (Chrome, Firefox, Safari, etc.).
3.  Cliquez n'importe où sur la page pour voir l'effet !
